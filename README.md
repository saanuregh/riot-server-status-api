# Riot Server Status API

An unofficial Riot server status API to find the server status of your favourite Riot game across all regions. Implemented as a serverless function and hosted on Vercel â™¥. Data obtained unoffically from offical [Riot status](https://status.riotgames.com) website.

## API

| Data                 | URL                                    |
| -------------------- | -------------------------------------- |
| All games            | https://riotstatus.vercel.app          |
| Valorant             | https://riotstatus.vercel.app/valorant |
| Legends of Runeterra | https://riotstatus.vercel.app/lor      |
| League of Legends    | https://riotstatus.vercel.app/lol      |
